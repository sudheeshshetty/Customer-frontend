# Customer Frontend

### Setup
##### Cloning the project
`git clone https://github.com/sudheeshshetty/Customer-frontend.git`

##### Steps

 - cd Customer-frontend
 - npm i

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
